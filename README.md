    This lab demonstrates essential Linux commands for navigating the filesystem, managing files and directories, viewing system information, and modifying permissions. Screenshots of command outputs are used to document and visualize their effects.

List of commands:

| Command      | Description                                   |
|--------------|-----------------------------------------------|
| pwd          | Prints current working directory             |
| ls           | Lists files and directories                  |
| ls -a        | Lists all files, including hidden ones      |
| ls -l        | Lists files in long format with details     |
| cd           | Changes the current directory               |
| mkdir        | Creates a new directory                      |
| rmdir        | Removes an empty directory                   |
| rm           | Deletes a file                               |
| rm -r        | Deletes a directory recursively             |
| touch        | Creates a new empty file                     |
| cat          | Displays file content                        |
| vi           | Opens a file in the vi editor               |
| cp           | Copies files or directories                  |
| mv           | Moves or renames files/directories           |
| locate       | Finds files by name                          |
| echo         | Prints text to the terminal                  |
| uname        | Displays system information                  |
| df           | Shows disk space usage                        |
| ps           | Lists running processes                       |
| top          | Shows real-time system processes             |
| chmod        | Changes file permissions                     |
